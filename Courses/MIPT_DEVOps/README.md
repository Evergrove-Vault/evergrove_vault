# Описание 

Курс предназначен для освоения базовых навыков работы с моделями машинного обучения ([[Термины и сокращения#Machine Learning (ML)|ML]]) в [[Термины и сокращения#Production environment (PROD)|PROD]] среде, а так же исследований на этапе их создания и проведения экспериментов над архитектурой и способами обучения. Студенты в рамках курса научатся:

- планировать эксперименты над моделями при помощи [[Термины и сокращения#HADI|HADI]] цикла
- создавать воспроизводимые пайплайны обучения и обработки данных 
- систематизировать результаты экспериментов и анализировать влияние параметров
- создавать распределенные пайплайны обучения и инференса моделей ML
- применять инструменты для ускорения [[Термины и сокращения#Runtime|runtime]] моделей
- работать с бачевым и потоковым инференсом 
- работать с потоками данных на PROD и строить пайплайны дообучения моделей на новых данных 
- прототипировать интерфейсы для запуска [[Термины и сокращения#Minimum viable product (MVP)|MVP]] решений на основе gradio, streamlit, fastapi
- строить системы логирования для отслеживания жизненного цикла моделей в PROD
