## Системы версионирования информации

### Git

Для начала надо разобраться с тем, как версионировать информацию, чтобы в дальнейшем точно знать что и когда использовалось и как обрабатывалось. Это очень важно в вопросе воспроизводимости любых действий. 

В качестве промышленного стандарта версионирования будем использовать git, как самую распространенную систему с большим количеством возможностей. 

В рамках курса мы научимся тонкостям работы с git и поймем его ограничения и недостатки.

### DVC

Основной проблемой git в работе с данными, а не кодовой базой часто является механизм индексации данных для проверки отсутствия изменений. Git не был изначально предназначен под тяжелые файлы, так что появилось решение в виде DVC, которое заточено под версионирование наборов данных и их обработку с помощью пайплайнов. 

По факту DVC является надстройкой над git и работает с ним в паре. Принципы работы и примеры использования описаны [[DVC|тут]]
